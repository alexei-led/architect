# architect

> Instruction-first architecture design and review package for AI coding runtimes. It ships a read-only Architect role, design/review/scorecard/plan skills, methodology guidance, report/design/plan templates, and helper CLIs for evidence-based architecture reports.

Use this file as a short map to the highest-signal project docs. The Architect designs, assesses, and plans; it does not mutate production source code. Establish intended architecture before scoring, cite evidence for every finding, record missing tools as confidence gaps, and prefer incremental refactoring with fitness checks.

Source templates and `src/templates/scorecard.yaml` are authoritative when prose docs disagree.
